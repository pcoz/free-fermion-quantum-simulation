# A free-fermion (matchgate) circuit, in the plain "NAME qubits" format, on a
# line of 16 qubits. EVERY gate is a matchgate acting on neighbouring qubits, so
# the interacting-gate count is k = 0 and the router picks FREE FERMION -- even
# though the circuit is highly non-Clifford (all those rotations) and builds
# real entanglement (the central bond is exercised every layer).
#
# Try it:   python simulator_router.py examples/free_fermion.txt
# Experiment: add a long-range or interacting line like `cx 0 8` (k -> 1) and
# watch the free-fermion cost tick up by a notch, exactly as in the demo.
qubits 16
